# Installation Guide

## Requirements

- PHP 8.0 or higher
- MySQL/MariaDB
- PDO MySQL extension
- Writable uploads directory
- Apache/Nginx with PHP enabled

## Installation Flow

1. Upload the project files to hosting or localhost.
2. Open `installer.php`.
3. Select business type.
4. Enter database credentials.
5. Create admin account.
6. Complete installation.
7. Delete or protect the installer after installation.

## After Installation

Open:

- `/admin/erp/production-hardening-dashboard.php`
- `/admin/erp/table-column-checker.php`
- `/admin/erp/settings-repair.php`
- `/admin/erp/permission-repair.php`
- `/admin/erp/demo-data-manager.php`
