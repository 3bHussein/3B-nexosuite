# User Manual

## Daily Workflow

1. Create or import products.
2. Add customers and suppliers.
3. Create quotations, orders and invoices.
4. Record payments.
5. Track inventory movement.
6. Use document library for contracts, licenses and files.
7. Use reports and dashboards for management follow-up.

## Portals

- Customer portal: service requests and customer interaction.
- Employee portal: internal access and employee workflow.
- Technician mobile: job card and service execution foundation.
